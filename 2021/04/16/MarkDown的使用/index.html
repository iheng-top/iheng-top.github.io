MarkDown的使用

发表于 2021-04-16 分类于 教程 阅读次数：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

Markdown 语言在 2004 由约翰·格鲁伯（John Gruber）创建。

Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

Markdown 编写的文档后缀为 .md, .markdown。

2、字体效果

  • 斜体

    语法：*[斜体的文字]*

    效果：斜体的文字

  • 加粗

    语法：**[加粗的文字]**

    效果：加粗的文字

  • 斜体 + 加粗

    语法：***[斜体加粗的文字]***

    效果：斜体加粗的文字

  • 删除线

    语法：~~[删除的文字]~~

    效果：删除的文字

  • 下划线

    语法：\加下划线的文字\

    效果：加下划线的文字
